# Plantillas de Ejercicios

Esta carpeta contiene las plantillas de los archivos que necesitarás crear para completar el curso.

## ¿Qué hacer con estas plantillas?

Sigue las instrucciones en **INSTRUCCIONES_CONFIGURACION.md** (en la raíz del proyecto) para configurar tu entorno de trabajo.

## Estructura de Plantillas

```
plantillas/
├── datos/
│   └── descargar_datos.py      # Script para descargar el dataset
└── ejercicios/
    ├── 01_cargar_sqlite.py     # Ejercicio 1: Bases de datos SQLite
    ├── 02_limpieza_datos.py    # Ejercicio 2: Limpieza con Pandas
    ├── 03_parquet_dask.py      # Ejercicio 3: Big Data con Dask
    └── 04_pyspark_query.py     # Ejercicio 4: Apache Spark
```

## Importante

**NO modifiques los archivos en esta carpeta.**

Copiarás estos archivos a las carpetas `datos/` y `ejercicios/` que crearás en la raíz del proyecto.

Ver: **INSTRUCCIONES_CONFIGURACION.md** para el paso a paso.
